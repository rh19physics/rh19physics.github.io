1   Equations

Inline math

Use \( ... \) for inline math. For example, the energy is \(E = \frac{1}{2}mv^2\), the Boltzmann factor is \(e^{-\beta \varepsilon}\), and a gradient is \(\nabla \cdot \vb{B} = 0\).

Display equations (unnumbered)

Use \[ ... \] for a centred, unnumbered equation:

\[ \frac{\pd f}{\pd t} + \vb{v}\cdot\nabla f + \frac{q}{m}\!\left(\vb{E}+\vb{v}\times\vb{B}\right)\cdot\nabla_v f = \left(\frac{\pd f}{\pd t}\right)_{\!c} \]

Numbered equations

Wrap in \begin{equation} ... \end{equation} for auto-numbering. You can label and reference them.

\begin{equation} \nabla\times\vb{E} = -\frac{\pd \vb{B}}{\pd t} \label{eq:faraday} \end{equation}

Reference it like this: Faraday's law is Eq. \(\eqref{eq:faraday}\).

Multi-line / aligned equations

Use \begin{align} ... \end{align} (each line numbered) or align* (no numbers). Use & for alignment and \\ for line breaks.

\begin{align} \nabla \cdot \vb{E} &= \frac{\rho}{\varepsilon_0} \label{eq:gauss} \\ \nabla \cdot \vb{B} &= 0 \label{eq:divB} \\ \nabla \times \vb{E} &= -\frac{\pd \vb{B}}{\pd t} \label{eq:faradayM}\\ \nabla \times \vb{B} &= \mu_0\vb{J} + \mu_0\varepsilon_0 \frac{\pd \vb{E}}{\pd t} \label{eq:ampere} \end{align}

Maxwell's equations: \(\eqref{eq:gauss}\)–\(\eqref{eq:ampere}\).

Cases / piecewise

\[ f(x) = \begin{cases} 1 & x > 0 \\ 0 & x = 0 \\ -1 & x < 0 \end{cases} \]

Matrices

\[ \mathsf{T} = \begin{pmatrix} T_{xx} & T_{xy} & T_{xz} \\ T_{yx} & T_{yy} & T_{yz} \\ T_{zx} & T_{zy} & T_{zz} \end{pmatrix} \]

Custom macros

Macros defined in the MathJax config at the top of this file: \vb{v} → \(\vb{v}\), \pd → \(\pd\), \dd → \(\dd\), \avg{x} → \(\avg{x}\). Add your own in the macros object.

4   Theorems & proofs

Theorem 1 (Parseval's theorem).

If \(f(x)\) is square-integrable on \([-L, L]\) with Fourier coefficients \(c_n\), then \[ \frac{1}{2L}\int_{-L}^{L} |f(x)|^2\,\dd x = \sum_{n=-\infty}^{\infty} |c_n|^2. \]

Proof.

Expand \(|f|^2 = f\bar{f}\) in Fourier series and integrate term by term, using orthogonality of \(e^{i n\pi x/L}\).

6   Code

Inline code

Use <code>...</code> for inline snippets like np.fft.fft(data).

Code blocks

import numpy as np

def energy_spectrum(B, dx):
    """Compute the magnetic energy spectrum."""
    Bk = np.fft.rfft(B) * dx
    k  = np.fft.rfftfreq(len(B), d=dx) * 2 * np.pi
    Ek = 0.5 * np.abs(Bk)**2 / len(B)
    return k, Ek

8   Footnotes

The Vlasov equation1 describes the evolution of the distribution function in the collisionless limit. It can be derived from the BBGKY hierarchy2.

  1. Strictly, the Vlasov equation is the collisionless Boltzmann equation. ↩
  2. Bogoliubov–Born–Green–Kirkwood–Yvon hierarchy; see Ichimaru (1973). ↩
