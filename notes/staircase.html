A Staircase Puzzle and the Fibonacci Numbers

Puzzle note · Mathematics

Imagine a staircase with 10 steps. Each move you make can climb either one step or two steps. How many different ways are there to reach the top? What if the staircase has 100 steps instead?

Start with small cases

If the staircase has 1 step, there is only one way to climb it. If the staircase has 2 steps, there are two ways:

  • 1 + 1
  • 2

Continuing this process gives the following pattern.

Steps Ways
11
22
33
45
58

These numbers form the Fibonacci sequence.

Why Fibonacci appears

Suppose the staircase has \(n\) steps. The final move must be either

  • a single step
  • a double step

If the last move is one step, the previous position must be step \(n-1\). If the last move is two steps, the previous position must be step \(n-2\). Therefore

\[ F(n) = F(n-1) + F(n-2) \]

which is exactly the defining rule of the Fibonacci sequence.

A small FORTRAN program

program fibo

implicit none

integer :: i, j
integer, allocatable :: n(:)

print *, "input the index"
read *, i

allocate(n(i))

n(1) = 1
n(2) = 2

do j = 3, i
n(j) = n(j-1) + n(j-2)
end do

print *, n(i)

end program fibo
